HTML

Es el lenguaje estándar para crear y estructurar páginas web. Utiliza etiquetas para definir elementos como texto, imágenes, enlaces y tablas. Es la base de cualquier sitio web y se complementa con CSS y JavaScript para mejorar su diseño y funcionalidad.

Css

Es un lenguaje de hojas de estilo que permite dar formato y diseño a las páginas web creadas con HTML. Con CSS, se pueden modificar colores, fuentes, márgenes, tamaños, disposición de los elementos y adaptabilidad a diferentes dispositivos (diseño responsivo).

JavaScript

Es un lenguaje de programación que permite agregar dinamismo e interactividad a las páginas web. A diferencia de HTML y CSS, que definen la estructura y el diseño, JavaScript puede modificar el contenido en tiempo real, responder a eventos del usuario (como clics o teclas presionadas) y realizar peticiones a servidores para actualizar información sin recargar la página (usando AJAX o Fetch API). JavaScript es compatible con todos los navegadores y se usa tanto en el frontend (con frameworks como React o Vue) como en el backend (con Node.js).

Frameworks

Son entornos de desarrollo que proporcionan una estructura predefinida para facilitar la creación de aplicaciones. Incluyen herramientas, librerías y reglas que optimizan la organización del código y aceleran el desarrollo.

En el desarrollo web, existen frameworks para diseño (como Bootstrap o Tailwind CSS), interactividad (como React, Angular o Vue.js) y backend (como Django, Laravel o Express.js). Su uso mejora la eficiencia, la escalabilidad y el mantenimiento de los proyectos.
